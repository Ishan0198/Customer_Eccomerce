# Customer E-Commerce Analytics Project

## Project Overview

This project demonstrates an end-to-end E-Commerce Analytics solution designed to transform raw transactional data into actionable business insights. The solution leverages SQL, Power BI, and data modeling techniques to analyze customer purchasing behavior, sales performance, product trends, and business KPIs.

The objective of the project is to help business stakeholders monitor revenue growth, customer engagement, product performance, and operational efficiency through interactive dashboards and analytical reporting.

## Business Problem

E-commerce businesses generate large volumes of transactional data from customers, products, orders, and payments. However, without proper analytics, identifying revenue drivers, customer retention opportunities, and product performance trends becomes challenging.

This project addresses these challenges by creating a centralized analytical solution that enables business users to:

* Track sales and revenue performance
* Analyze customer purchasing behavior
* Monitor top-performing products and categories
* Evaluate order and payment trends
* Support data-driven decision making

## Dataset Components

The project utilizes multiple interconnected datasets, including:

* Customers
* Orders
* Order Details
* Products
* Categories
* Payments
* Shipping Information

These datasets are modeled using a star schema approach to optimize analytical performance and reporting efficiency.

## Key Features

### Data Modeling

* Designed relationships between fact and dimension tables.
* Implemented a scalable star schema model.
* Applied data cleansing and transformation techniques.

### SQL Analytics

* Developed complex SQL queries using:

  * Joins
  * CTEs
  * Window Functions
  * Aggregations
  * Ranking Functions
  * Business KPI calculations

### Power BI Dashboard

* Built interactive dashboards with drill-down and filtering capabilities.
* Developed DAX measures for advanced business calculations.
* Created KPI cards, trend analysis, and comparative reporting visuals.

### Business KPIs
